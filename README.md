# Nexura

A premium dark-themed creative agency landing page designed for high-performance ecommerce brands.

## Overview

Nexura is a modern, responsive marketing website focused on showcasing creative services, performance metrics, testimonials, workflows, and conversion-focused call-to-actions.

Built with a sleek futuristic design, smooth animations, glassmorphism effects, and a fully responsive layout.

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/nexura.git
```

2. Open the project folder

```bash
cd nexura
```

3. Launch the website

Simply open:

```bash
index.html
```

Or use a local server:

```bash
python -m http.server 8000
```
